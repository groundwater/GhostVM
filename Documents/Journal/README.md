# Journal Standard

Every agent run adds one entry at:

`Documents/Journal/YYYY/MM/DD/{title}.md`

Use a short kebab-case title.

## Template

```md
# <Title>

Date: YYYY-MM-DD
Agents: <who ran>
Scope: <what this run changed or investigated>

## Intent

<why this run happened>

## Actions

- <key action>
- <key action>

## Verification

- <command or check>

## Outcome

<result and any follow-up>

## Next Questions

- <open question or `None.`>
```
